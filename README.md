# CuraLink - Healthcare Application

A comprehensive full-stack healthcare platform connecting patients with doctors for appointments, symptom checking, ambulance services, and home visits.

## Features

### Patient Portal
- **User Authentication**: Secure login and registration with JWT
- **Dashboard**: Welcome message, upcoming appointments, recent analysis, quick actions
- **Symptom Checker**: AI-powered symptom analysis with voice recording support
- **Appointment Booking**: Book appointments with specialist doctors
- **Doctor Selection**: Filter doctors by specialization
- **Emergency Ambulance**: 24/7 ambulance booking with live location tracking
- **Home Visit Requests**: Request doctor home visits
- **Prescriptions**: View and manage prescriptions
- **Profile Management**: Update personal information

### Doctor Portal
- **Doctor Dashboard**: View patient statistics and pending requests
- **Appointment Management**: Accept, confirm, or complete appointments
- **Patient Details**: View comprehensive patient information
- **Prescriptions**: Add and manage prescriptions
- **Home Visit Requests**: Accept or decline home visit requests
- **Ambulance Tracking**: Track emergency ambulance locations during emergencies

### System Features
- **JWT Authentication**: Secure token-based authentication
- **Password Security**: Bcrypt password hashing
- **SQLite Database**: Local database for development
- **Responsive Design**: Mobile-friendly interface
- **Error Handling**: Comprehensive error handling and validation
- **Real-time Updates**: Simulated real-time tracking for ambulances
- **Pre-seeded Data**: Default doctor accounts for testing
